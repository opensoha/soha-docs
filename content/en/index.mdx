# OpenSoha Docs

This site is the public documentation and product entry for OpenSoha.

`soha-docs` is no longer just a set of internal notes split out from the core repository. It is the public documentation site for OpenSoha:

- Explain OpenSoha, Soha, the self-hosted edition, and public capability boundaries for new users.
- Provide deployment, configuration, Agent Runtime, AI Gateway, MCP, and governance documentation for self-hosted users.
- Provide `soha-contracts`, OpenAPI, SDK, plugin manifest, and cross-repository boundary references for developers.

## Public capability boundary

The public docs focus on the open-source, self-hosted distribution: Server API, Agent protocol, AI Gateway, MCP / Skills, plugin extension interfaces, local authentication, RBAC, audit, resource management, and single-binary delivery.

Documentation content is limited to public APIs, contracts, SDKs, artifacts, and extension interfaces.
