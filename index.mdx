## 这个站点现在承担什么

`soha-docs` 不再只是从 core 仓库拆出来的一组内部文档。它是 OpenSoha 的公开文档站和产品入口：

- 面向新用户解释 OpenSoha、Soha、自托管版本和 Soha Cloud 的关系。
- 面向自托管用户提供部署、配置、Agent Runtime、AI Gateway、MCP 和治理文档。
- 面向 SaaS 用户提供 Soha Cloud 登录入口；真实 Cloud 地址通过 `SOHA_CLOUD_LOGIN_URL` 在构建时注入。
- 面向开发者提供 `soha-contracts`、OpenAPI、SDK、插件 manifest 和跨仓库边界说明。

## 开源 / SaaS 边界

开源 `soha` core 负责单租户或默认租户控制平面、Server API、Web 控制台集成点、Agent 协议、基础 AI Gateway、本地鉴权、RBAC、审计、资源管理和单二进制交付。

Soha Cloud 负责多租户生命周期、组织、工作区、成员邀请、SaaS IAM、SSO、套餐、额度、计费、云端运营后台、托管连接器和托管 Agent Fleet。Cloud-only 逻辑保留在闭源 `soha-cloud`，通过公开 API、contracts、SDK、artifact 或扩展接口复用 core 能力。
