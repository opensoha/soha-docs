# Roadmap

## MVP Scope

- multi-cluster bootstrap and health summary
- namespace, pod, and deployment browsing
- dashboard shell and navigation system
- basic audit log persistence
- documentation site and engineering guide
- MCP integration placeholder contracts

## Current Implemented Baseline

- cluster inventory and cluster detail diagnostics
- workloads, network, and storage resource views
- access control read/write management baseline
- application registry and manual build trigger
- release center MVP with deployment image update
- alert inventory, notification channels, alert routes, and downstream fan-out
- AI inspection tasks, inspection runs, and assistant-linked inspection suggestions

## Phase 2

- richer workload actions
- logs, exec, and rollout history
- release approval gates and promotion workflow
- Prometheus, Alertmanager, Grafana, and Argo CD adapter layer
- event center enrichment and correlation views

## Phase 3

- notifications and approval workflow
- CI/CD deep integrations
- broader MCP capability surface
- policy simulation and access review workflows
- external model gateway and tool-augmented AI automation
